This function implements the beahvior of the robot when the action check\+\_\+consistency is executed. This action is performed after a new complete hypothesis has been found and the robot reaches home in order to verify the consistency of the current complete hypothesis. In particular this is done by two clients to the services comm and consistent, in particular this last service is composed by a boolean response that is set to true when the compelete hypothesis is also consistent and is set to false when it is not. Of course if the response is false then a replanning is neeeded.
