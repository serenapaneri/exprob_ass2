Description\+: ~\newline
 In this node all the services of the rosplan are called in order to execute the current plan generated automatically, thanks to those services, from the domain and the problem files provided. Since all those services are called within a loop, everytime that the plan\+\_\+dispatcher service returns that the plan has failed, the replanning is automatically started.
