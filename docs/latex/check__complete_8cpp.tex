 In this node is implemented the check\+\_\+complete action of the rosplan. This node allows to know if a new complete hypothesis has been found, and this is perfomed thanks to the complete service that advertise this node. If a new complete hypothesis has been found, then the plan can proceed, if instead there is no new complete hypothesis then the action return false and a replanning is needed.
