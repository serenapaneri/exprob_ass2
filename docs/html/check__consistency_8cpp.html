Description:
In this node is implemented the check_consistency action of the rosplan. This node allows to know if an hypotheis is consistent or not and this is perfomed thanks to the consistent service that advertise this node. If the hypothesis is complete and consistent then the plan can proceed, if instead it is inconsistent then a replanning is needed.
